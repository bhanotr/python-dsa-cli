Hint Level 1:

General approach:
- Use a hash map (dictionary) to group anagrams
- Sort each string to create a key
- All anagrams have the same sorted string

Key concept:
- Sorted string as hash map key
- "eat" -> "aet", "tea" -> "aet", "ate" -> "aet"
- Use "aet" as key to group these together

Hash map operations:
```python
from collections import defaultdict
groups = defaultdict(list)

# Add to group
key = ''.join(sorted(s))
groups[key].append(s)
```

Hint Level 2:

Detailed approach for group_anagrams():
1. Create hash map (defaultdict(list)) for groups
2. Iterate through each string in strs
3. Sort the string to create a key
4. Add the original string to the group for that key
5. Return list of all groups

Example:
```python
strs = ["eat", "tea", "tan", "ate", "nat", "bat"]

"eat" -> sorted = "aet" -> groups["aet"] = ["eat"]
"tea" -> sorted = "aet" -> groups["aet"] = ["eat", "tea"]
"tan" -> sorted = "ant" -> groups["ant"] = ["tan"]
...
```

String sorting:
```python
sorted_str = ''.join(sorted("eat"))  # "aet"
```

=== HINT LEVEL ===

Hint Level 3:

Complete solution:
```python
from typing import List
from collections import defaultdict

def group_anagrams(strs: List[str]) -> List[List[str]]:
    groups = defaultdict(list)
    
    for s in strs:
        key = ''.join(sorted(s))
        groups[key].append(s)
    
    return list(groups.values())
```

Time complexity: O(n * k log k) where n is number of strings, k is max string length
Space complexity: O(n * k) - Store all strings in hash map

Alternative (character count - better for long strings):
```python
def group_anagrams(strs: List[str]) -> List[List[str]]:
    groups = defaultdict(list)
    
    for s in strs:
        count = [0] * 26
        for c in s:
            count[ord(c) - ord('a')] += 1
        key = tuple(count)
        groups[key].append(s)
    
    return list(groups.values())
```
This is O(n * k) time!
