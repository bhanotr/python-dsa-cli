Hint Level 2:

Detailed approach for is_palindrome():
1. Initialize left = 0, right = len(s) - 1
2. While left < right:
   a. Skip non-alphanumeric from left
   b. Skip non-alphanumeric from right
   c. Compare lowercase characters
   d. If different, return False
   e. Move both pointers inward
3. Return True if loop completes

=== HINT LEVEL ===

Hint Level 3:

Complete solution:
```python
def is_palindrome(s: str) -> bool:
    left, right = 0, len(s) - 1
    
    while left < right:
        # Skip non-alphanumeric from left
        while left < right and not s[left].isalnum():
            left += 1
        
        # Skip non-alphanumeric from right
        while left < right and not s[right].isalnum():
            right -= 1
        
        # Compare characters
        if s[left].lower() != s[right].lower():
            return False
        
        left += 1
        right -= 1
    
    return True
```

Time complexity: O(n) - Single pass
Space complexity: O(1) - No extra space

Alternative (clean and compare):
```python
def is_palindrome(s: str) -> bool:
    cleaned = ''.join(c.lower() for c in s if c.isalnum())
    return cleaned == cleaned[::-1]
```
This uses O(n) space but is simpler.
